The plugin by PushLoop to enable automated Push Notifications. Engage Mobile Users Without an App!

== Description ==
The pushloop plugin to enable automated push notifications by Pushloop. Engage mobile users without an app!
Visitors who choose to receive pushes will receive them even if they are not on the site, thus encouraging them to return to your page!

The <a href="https://pushloop.io">PushLoop</a> plugin for WordPress Web Push Notifications automates the integration.

You can send notifications at preset times or you can send notifications to followers every time you publish a new post.

Our plugin is based on our <a href="https://app.pushloop.io/openapi.json">API</a> available in OpenAPI format.

Here you can find more information about our <a href="https://www.pushloop.io/termini-e-conditioni-duso/">Terms and Conditions</a>.

== Usage of Third-Party Service ==
This plugin uses the push notification service provided by <a href="https://amp.dev/">AMP</a> for systems using AMP. Integration with the service occurs when users enable AMP functionality from the plugin.
Here you can find more information about their <a href="https://terms-of-use.openjsf.org/">Terms of Use</a> and <a href="https://privacy-policy.openjsf.org/">Privacy Policy</a>.

== Installation ==
1. Install Pushloop from the WordPress.org plugin market or by uploading the Pushloop plugin folder into your WP-site plugins directory.
2. Activate the Pushloop plugin from your WordPress settings dashboard.
3. Configure your site from the configuration section of the plugin.

== Configuration ==
Add the Website ID and the Rest API Key from the "Manage sites" section of your PushLoop dashboard.
